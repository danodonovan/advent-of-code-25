# Getting started

Check out https://adventofcode.com/2025/

## New Puzzle

0. Create `day-XX` branch from `main`
0. Create folder `day-XX` to work in

### for Zig
0. In that directory run `$ zig init` to create a new Zig project

### for Python
0. Check virtualenv, install [`requirements.txt`](requirements.txt) if needed
0. Create `solution.py` file
